# TAZITA'S SHOP

## Sobre el proyecto
** Sitio ecommerce de productos para pasteleria llamado Tazita's Shop, realizado como proyecto final para el curso de React JS en CoderHouse. **

#### Dependencias utilizadas
- Material UI
- React Router DOM
- Firebase
